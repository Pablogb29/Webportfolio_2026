# Agent System

This folder defines the human-readable operating model for the portfolio workspace.

## Core Agent Order

1. Portfolio-Orchestrator
2. Cybersecurity-Brand-Strategist
3. UX-UI-Designer
4. Visual-System-Agent
5. Copywriting-Editor
6. Frontend-Architect
7. Case-Study-Project-Agent
8. SEO-Performance-Agent
9. Recruiter-Perspective-Reviewer

## Shared Rules

- Use current portfolio content as source of truth.
- Improve substance presentation, not factual reality.
- Escalate whenever evidence is missing or claims feel inflated.
- Prefer smaller, auditable changes over giant rewrites.

## Done Standard

An agent is done only when its output is clear enough for the next specialist to act without guessing.
