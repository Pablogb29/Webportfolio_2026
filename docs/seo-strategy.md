# SEO Strategy

## Search Intent To Support

- Cybersecurity engineer portfolio
- Pentester portfolio
- Active Directory security projects
- IAM automation engineer
- Hack The Box writeups
- OSCP-oriented portfolio or labs

## SEO Principles

- Each route needs a unique search purpose.
- Title tags should combine role, specialty, and evidence.
- Meta descriptions should sound human and specific.
- Headings should mirror how recruiters search and skim.
- Internal links should connect related proof: projects, writeups, case studies, certifications.

## Structured Data Priorities

- Person
- WebSite
- BreadcrumbList
- Article for future writeups and case studies

## Homepage Metadata Focus

Emphasize cybersecurity engineering, penetration testing trajectory, Active Directory focus, IAM/security automation, and documented practical work.

## Content SEO Rules

- Do not force keywords into every paragraph.
- Prefer exact technical language over generic growth copy.
- Use descriptive slugs.
- Make public project links and writeups crawlable when appropriate.

## Performance-SEO Connection

Fast pages, clean semantic structure, stable layout, and mobile readability are part of SEO, not separate work.
