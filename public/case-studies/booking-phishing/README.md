# Booking Phishing Case Study - Image Assets

This directory contains images extracted from the PDF report `Report_ES_Booking_Scam_Luxembourg.pdf`.

## Image Naming Convention

- `fig-01.png` - Hero image (main case study image)
- `fig-02.png` - Suspicious Facebook profile
- `fig-03.png` - Broken Booking.com link
- `fig-04.png` - Phishing website interface
- `fig-05.png` - Domain WHOIS information
- `fig-06.png` - Payment collection form
- `fig-07.png` - Post-24h executable download attempt

## Extraction Instructions

To extract images from the PDF:

1. Open `Report_ES_Booking_Scam_Luxembourg.pdf` in a PDF viewer
2. Export each figure/screenshot as PNG
3. Rename according to the convention above
4. Place all images in this directory

## Image Requirements

- Format: PNG
- Recommended resolution: 1920x1080 or higher for hero image
- Other figures: Minimum 800x600
- Optimize images for web (reduce file size while maintaining quality)
